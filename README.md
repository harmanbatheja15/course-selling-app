# Saas Platform for Educators

<br/>

# Technologies Used:

React, Node, Express, Postgres, Prisma ORM, Tailwind CSS

# Steps to run the project:

-   git clone `https://github.com/harmanbatheja15/course-selling-app.git`

## Backend:

-   cd `backend`
-   npm install
-   cp .env.example .env
-   npm run dev

## Frontend:

-   cd `frontend`
-   npm install
-   cp .env.example .env
-   npm run dev
